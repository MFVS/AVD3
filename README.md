## Návod k použití

Obrázky map jsou generovány v `podle_mest.ipynb` a `podle_statu.ipynb`.

Prerekvizita ke spuštění je python.
Aplikace se spouští příkazy `pip install -r requirements.txt` a `streamlit run streamlit.py`.
